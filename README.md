# Case - QA Automation & Load Testing Suite

## 📋 Repository Overview

This repository contains two main testing projects:
1. **Task1-UITest** - Selenium-based UI automation testing for Insider careers page
2. **Task2-LoadTest** - Apache JMeter load testing for e-commerce platform

3. **Task3-APITest** - API tests for petstore.swagger.io with JUnit and Jackson.

Please go to each project's folder to see details.
